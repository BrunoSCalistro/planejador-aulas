⚙️ Configurar Google Drive
Seus dados ficam salvos na sua conta Google, com segurança
Você precisará criar um projeto gratuito no Google Cloud Console e colar o Client ID abaixo. Siga os passos:
1
Acesse o Google Cloud Console
Abra em outra aba: console.cloud.google.com e faça login com sua conta Google.
2
Crie um projeto
Clique em "Novo Projeto", dê um nome (ex: planejador-aulas) e clique em Criar.
3
Ative a Google Drive API
No menu lateral, vá em APIs e Serviços → Biblioteca. Busque por Google Drive API e clique em Ativar.
4
Configure a Tela de Consentimento OAuth
Vá em APIs e Serviços → Tela de consentimento OAuth. Escolha Externo, preencha o nome do app (ex: Planejador de Aulas) e seu e-mail. Salve. Na aba Escopos, adicione ../auth/drive.file. Na aba Usuários de teste, adicione seu próprio e-mail.
5
Crie as Credenciais OAuth
Vá em APIs e Serviços → Credenciais → Criar Credenciais → ID do cliente OAuth. Escolha Aplicativo da Web. Em "Origens JavaScript autorizadas" adicione null (para arquivo local). Clique em Criar e copie o ID do cliente.
Será salvo localmente apenas neste navegador.
