⚙️ Configurar Google Drive
Seus dados ficam salvos na sua conta Google
Você precisará de um projeto no Google Cloud Console. Siga os passos:
1
Acesse o Google Cloud Console
console.cloud.google.com — faça login com sua conta Google.
2
Crie um projeto
Clique em "Novo Projeto", nomeie (ex: planejador-aulas) e crie.
3
Ative a Google Drive API
APIs e Serviços → Biblioteca → busque Google Drive API → Ativar.
4
Crie credenciais OAuth
Credenciais → Criar → ID do cliente OAuth → Aplicativo da Web. Em "Origens JavaScript autorizadas" adicione a URL do seu app (ex: https://brunoscalistro.github.io). Copie o Client ID.
Salvo localmente neste navegador.
